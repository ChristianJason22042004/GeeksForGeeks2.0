🐳 Run with Docker
1️⃣ Build Image
docker build -t geekforgeeks-clone:latest .

2️⃣ Run Container
docker run -d -p 8080:80 --name priceless_wilber geekforgeeks-clone:latest

3️⃣ Access Project

👉 http://localhost:8080/index.html
